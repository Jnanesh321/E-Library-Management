# E-Library Management System

A full-stack web application that allows users to access digital books online.

## Features
- User authentication
- Upload and manage e-books (PDF)
- Search and filter books
- Read/download books
- Role-based access (Admin, Student)

## Tech Stack
- Backend: Node.js + Express
- Database: MongoDB
- Frontend: React / EJS
